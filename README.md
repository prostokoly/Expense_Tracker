Expense Tracker
Персональное приложение для учета расходов и доходов с современным интерфейсом.
Технологии.

Frontend:
- React.js
- CSS3
- Axios для API запросов
  
Backend:
- Node.js
- Express.js
- Sequelize ORM
- PostgreSQL

  
Запуск проекта:
1. Установка зависимостей клиента
cd client,
npm install
2. Установка зависимостей сервера
cd server,
npm install
3. Запуск сервера
npm run dev
4. Запуск клиента 
npm run dev
5. Открыть http://localhost:3000
